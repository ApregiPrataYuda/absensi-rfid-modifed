Cara Menghubungkan Bot Telegram ke Sistem Absensi

Panduan ini dibuat untuk admin sekolah. Alurnya dimulai dari membuat bot lewat @BotFather, menyimpan token ke pengaturan aplikasi, sampai memberi instruksi ke siswa agar mereka menghubungkan akun Telegram masing-masing dengan /start NISN.

Webhook otomatis Diset saat token Telegram disimpan
Butuh HTTPS Agar Telegram bisa mengakses webhook aplikasi

Siapkan Dulu

  • Akun Telegram admin sekolah.
  • Data siswa dan NISN yang sudah benar.
  • Aplikasi sudah online dan memakai HTTPS.
  • Akses ke menu Pengaturan Notifikasi.

Target Akhir

  • Bot Telegram aktif dan valid.
  • Webhook berhasil dipasang ke aplikasi.
  • Siswa menghubungkan akun Telegram dengan NISN.
  • Notifikasi absensi terkirim ke Telegram.

Ringkasan Awal

Kalau ingin melihat gambaran cepat sebelum mulai, bagian ini cukup untuk memahami alurnya.

1. Buat Bot

  • Buat bot melalui @BotFather.
  • Simpan token yang diberikan Telegram.

2. Simpan Token

  • Masuk ke pengaturan Telegram di aplikasi.
  • Tempel token lalu simpan.

3. Webhook Aktif

  • Sistem akan validasi bot.
  • Webhook dipasang otomatis jika server sudah online.

4. Siswa Start

  • Siswa chat pribadi ke bot.
  • Kirim perintah /start NISN.
  • Jika satu akun Telegram dipakai untuk beberapa anak, ulangi untuk setiap NISN.
Hal penting sebelum mulai Jika aplikasi masih berjalan di localhost atau belum memakai HTTPS, Telegram tidak akan bisa mengakses webhook. Dalam kondisi itu token bisa saja tersimpan, tetapi status webhook belum akan normal.

Langkah Setting

Ikuti urutan di bawah ini dari awal. Jangan lompat ke langkah siswa sebelum token bot tersimpan dengan benar.

1

Buat bot baru di BotFather

BotFather adalah akun resmi Telegram untuk membuat bot baru.

  1. Buka Telegram lalu cari akun @BotFather.
  2. Kirim /start.
  3. Kirim /newbot.
  4. Masukkan nama bot, misalnya Absensi SMP Harapan.
  5. Masukkan username bot yang diakhiri bot, misalnya absensi_smp_harapan_bot.
  6. BotFather akan mengirim bot token. Simpan token ini.
Contoh token 1234567890:AAExampleTokenDariBotFather123456789
2

Masukkan token bot ke aplikasi

Langkah ini dilakukan dari dashboard admin sekolah.

  1. Login ke dashboard aplikasi.
  2. Buka menu Pengaturan Notifikasi.
  3. Masuk ke tab Konfigurasi Notifikasi.
  4. Pilih sub-tab Konfigurasi Telegram.
  5. Tempel token ke kolom Bot Token.
  6. Klik Simpan Pengaturan.
3

Biarkan sistem memvalidasi bot dan memasang webhook

Setelah disimpan, sistem akan bekerja otomatis.

  • Memeriksa apakah token valid.
  • Mengambil Bot ID dan Username Bot.
  • Menyinkronkan daftar command bot ke menu Telegram.
  • Memasang webhook Telegram ke aplikasi.
  • Memperbarui Webhook Status di halaman pengaturan.
Jika token benar Bot ID dan username akan terisi otomatis.
Jika server online Webhook bisa langsung aktif.
Jika ada masalah Cek status dan kolom error terakhir.
4

Aktifkan channel notifikasi Telegram

Bot sudah aktif saja belum cukup. Channel Telegram juga harus dipilih di pengaturan notifikasi.

  1. Pada kartu Notifikasi Absensi, aktifkan toggle notifikasi.
  2. Pilih Telegram atau WhatsApp + Telegram di dropdown Kirim via.
  3. Jika diperlukan, sesuaikan template pesan pada tab Template Pesan.

Perintah Bot

Setelah bot aktif, akun Telegram pengguna bisa menjalankan beberapa perintah berikut di chat pribadi bot.

/start NISN

  • Untuk menautkan akun Telegram ke data siswa.
  • Bisa juga dipakai dari deep link Telegram sekolah.

/cek

  • Menampilkan daftar siswa yang tertaut ke akun Telegram ini.
  • Berguna untuk orang tua dengan lebih dari satu anak.

/profil NISN

  • Menampilkan profil siswa yang tertaut ke akun Telegram ini.
  • Contoh: /profil 2432500012.

/absensi NISN

  • Menampilkan absensi terakhir siswa yang tertaut.
  • Contoh: /absensi 2432500012.

/izin NISN

  • Menampilkan data izin atau sakit terakhir siswa yang tertaut.
  • Contoh: /izin 2432500012.

/hapus NISN

  • Untuk melepas satu tautan siswa tertentu.
  • Contoh: /hapus 2432500012.

/hapussemua

  • Untuk melepas semua siswa dari akun Telegram ini.
  • Dipakai jika ingin reset semua tautan.

/menu

  • Menampilkan panduan command kembali.
  • Juga akan muncul otomatis jika user mengetik keyword yang salah.

Instruksi untuk Siswa

Bagian ini penting. Telegram tidak bisa mengirim ke siswa kalau siswa belum menghubungkan chat mereka ke sistem.

5

Siswa wajib chat pribadi ke bot

Instruksi ini bisa dibagikan lewat wali kelas, grup sekolah, atau pengumuman internal.

  1. Siswa buka bot Telegram sekolah.
  2. Siswa menekan tombol Start.
  3. Siswa kirim perintah dengan format:
/start NISN

Contoh untuk siswa:

/start 2432500012
Setelah langkah ini berhasil Sistem akan menyimpan hubungan antara data siswa di database dengan chat Telegram siswa tersebut. Setelah itu siswa baru bisa menerima notifikasi dari bot.
Jika satu akun Telegram dipakai untuk beberapa siswa Kirim ulang perintah /start NISN untuk setiap siswa yang ingin ditautkan. Misalnya orang tua dengan dua anak dapat mengirim /start 2432500012 lalu /start 2432500099 dari akun Telegram yang sama.
6

Aturan yang perlu disampaikan ke siswa

Supaya tidak banyak gagal hubung, sampaikan poin-poin ini secara jelas.

  • NISN yang dikirim harus benar dan sesuai data sekolah.
  • Perintah harus dikirim di chat pribadi bot, bukan di grup Telegram.
  • Jika salah NISN, siswa harus mengulangi dengan data yang benar.
  • Jika bot sekolah diganti, siswa perlu melakukan /start NISN lagi ke bot baru.

Contoh Pesan yang Bisa Dibagikan ke Siswa

Bagian ini bisa langsung disalin oleh admin sekolah atau wali kelas.

Silakan hubungkan akun Telegram Anda ke sistem absensi sekolah. Langkah: 1. Buka bot Telegram sekolah. 2. Tekan Start. 3. Kirim perintah dengan format: /start NISN Contoh: /start 2432500012 Gunakan NISN Anda sendiri. Setelah berhasil, notifikasi absensi akan masuk ke Telegram Anda.

Cara Cek Hasil Setting

Gunakan daftar berikut untuk memastikan proses sudah selesai dengan benar.

Di pengaturan Telegram

  • Bot ID terisi.
  • Username Bot terisi.
  • Webhook Status menunjukkan status aktif atau normal.

Di data koneksi siswa

  • Jumlah siswa terhubung bertambah.
  • Siswa yang sudah /start NISN bisa ditemukan di data link Telegram.

Saat uji kirim

  • Lakukan absensi uji pada satu siswa yang sudah terhubung.
  • Pastikan pesan masuk ke Telegram siswa tersebut.

Jika gagal

  • Cek token bot.
  • Cek status webhook.
  • Cek apakah siswa sudah benar-benar kirim /start NISN.
Jika bot Telegram diganti Siswa lama biasanya perlu menghubungkan ulang ke bot baru. Jadi setelah pergantian bot, kirim ulang instruksi /start NISN ke siswa.

Pertanyaan Umum

Bagian ini untuk membantu admin saat ada pertanyaan dari operator atau wali kelas.

Apakah webhook harus manual diset di Telegram?

Tidak. Di sistem ini webhook akan dicoba set otomatis saat token bot disimpan.

Kenapa bot valid tapi siswa belum menerima notifikasi?

Biasanya karena siswa belum melakukan /start NISN, atau chat tersebut belum tersimpan ke sistem.

Apakah siswa bisa kirim /start tanpa NISN?

Bisa, tetapi sistem tidak akan bisa menghubungkannya ke data siswa yang benar. Format yang harus dipakai adalah /start NISN.

Apakah satu akun Telegram bisa dipakai untuk beberapa siswa?

Bisa. Gunakan akun Telegram yang sama, lalu kirim /start NISN satu per satu untuk setiap siswa yang ingin ditautkan.

Kalau aplikasi masih lokal apakah bisa dipakai?

Untuk penyimpanan token mungkin bisa, tetapi webhook Telegram memerlukan URL publik dan HTTPS agar sinkronisasi chat siswa berjalan normal.

Dokumen ini dapat dibuka langsung dari browser lokal maupun dari server sekolah. Jika diperlukan, admin bisa mencetak isi panduan ini atau menyalin bagian "Contoh Pesan yang Bisa Dibagikan ke Siswa".

# ABSENSI MODIFIED
Absensi Modified adalah sistem manajemen kehadiran karyawan yang dikembangkan untuk mendukung proses pencatatan absensi, monitoring, dan pelaporan kehadiran secara digital di lingkungan perusahaan.

## About Absensi Modified

**Absensi Modified** adalah aplikasi berbasis Laravel yang digunakan untuk mencatat dan memonitor kehadiran karyawan. Sistem ini memudahkan perusahaan dalam mengelola data absensi, memantau jam masuk dan pulang, serta menghasilkan laporan kehadiran secara cepat.

## Features

* Manajemen data karyawan
* Manajemen Gedung dan Lantai
* Absensi masuk dan pulang
* Monitoring kehadiran harian
* Rekap absensi berdasarkan periode
* Export laporan ke Excel
* Dashboard statistik kehadiran
* Role & Permission Management
* Riwayat aktivitas pengguna

## Technology Stack

* Laravel
* PHP
* MySQL
* Bootstrap / Tailwind CSS
* JavaScript
* Font Awesome

## Installation

```bash
git clone <repository-url>
cd absensi-modified

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
